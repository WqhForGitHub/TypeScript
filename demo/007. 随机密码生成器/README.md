# 随机密码生成器

一个使用纯 TypeScript 实现的随机密码生成器 CLI 工具。基于 Node.js `crypto` 模块的安全随机数生成器，确保密码的随机性和安全性。

## 功能特性

- 使用 `crypto.randomInt()` 生成密码学安全的随机数
- 自定义密码长度
- 可选字符集：小写字母、大写字母、数字、特殊符号
- 支持排除易混淆字符（如 `0O`, `1lI`）
- 批量生成多个密码
- 密码强度评估（0-100 分制）
- 保证密码至少包含每种已启用字符类型的一个字符
- Fisher-Yates 洗牌算法保证字符分布随机性

## 安装与构建

```bash
# 安装依赖
npm install

# 编译 TypeScript
npm run build

# 运行
npm start
```

## 使用方法

### 基本用法

```bash
# 生成 16 位默认密码（包含大小写字母、数字、符号）
npm start
```

### 自定义密码长度

```bash
# 生成 20 位密码
npm start -- -l 20
```

### 自定义字符集

```bash
# 生成不含特殊符号的 12 位密码
npm start -- -l 12 --no-symbols

# 仅使用小写字母和数字
npm start -- --no-uppercase --no-symbols

# 仅使用数字（PIN 码）
npm start -- -l 6 --no-lowercase --no-uppercase --no-symbols
```

### 排除易混淆字符

```bash
# 排除 0, O, 1, l, I 等易混淆字符
npm start -- -e "0O1lI"
```

### 批量生成

```bash
# 生成 5 个密码
npm start -- -c 5
```

### 密码强度评估

```bash
# 生成密码并显示强度评估
npm start -- -s

# 生成 20 位密码并评估强度
npm start -- -l 20 -s
```

## 命令行选项

| 选项               | 说明                     | 默认值 |
| ------------------ | ------------------------ | ------ |
| `-l, --length`     | 密码长度                 | 16     |
| `--no-lowercase`   | 不包含小写字母           | -      |
| `--no-uppercase`   | 不包含大写字母           | -      |
| `--no-numbers`     不包含数字               | -      |
| `--no-symbols`     | 不包含特殊符号           | -      |
| `-e, --exclude`    | 排除指定字符             | 无     |
| `-c, --count`      | 生成密码数量             | 1      |
| `-s, --strength`   | 显示密码强度评估         | -      |
| `-h, --help`       | 显示帮助信息             | -      |
| `-v, --version`    | 显示版本号               | -      |

## 密码强度评估规则

评分范围 0-100 分：

| 分数范围 | 等级 |
| -------- | ---- |
| 0-20     | 极弱 |
| 21-40    | 弱   |
| 41-60    | 一般 |
| 61-80    | 强   |
| 81-100   | 极强 |

评分细则：
- **长度评分**（上限 30 分）：每 4 位加 10 分
- **字符类型评分**（上限 80 分）：每种类型加 20 分（小写/大写/数字/符号）
- **分布均匀性**（上限 10 分）：字符唯一率 >70% 加 10 分，>50% 加 5 分

## 技术要点

- **安全随机数**：使用 `crypto.randomInt()` 而非 `Math.random()`，提供密码学级别的随机性
- **字符保证**：通过预分配策略确保密码至少包含每种已启用字符类型的一个字符
- **洗牌算法**：使用 Fisher-Yates 算法对生成的字符进行随机排列，保证分布均匀
- **零运行时依赖**：仅使用 Node.js 内置模块，无第三方依赖
